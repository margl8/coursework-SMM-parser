У меня ошибка. Что делать?

  1. Сделайте скриншот экрана. Для Windows надо нажать Alt+PrintScreen, для Mac Shift+Command+3
  2. Пришлите письмо на почту margl8@yandex.ru, вставьте скриншот в письмо и опишите проблему
  3. Частые проблемы:
    1. Парсер подписчиков может не работать из-за того, что администраторы сообщества закрыли доступ к разделу с участниками группы. Это не ошибка, а ограничение со стороны платформы ВКонтакте
    2. Сервис работает только с открытыми данными, поэтому функции срабатывают с закрытыми сообществами.
    3. Функция может не срабатывать, так как была указана ссылка на аккаунт пользователя. Сервис работает корректно только с сообществами ВКонтакте.

Получение ID

  1. Скопируйте ссылку на группу ВКонтакте, id которой вам нужно

  2. Вставьте ссылку в текстовое окно
  3. Нажмите кнопку «Отправить»

  4. Если происходит ошибка, проверьте правильность написания ссылки. Ссылка должна иметь вид: https://vk.com/group_name

Парсинг подписчиков

  1. Скопируйте ссылку на группу ВКонтакте, участники которой вам нужны

  2. Вставьте ссылку в текстовое окно
  3. Нажмите кнопку «Отправить»

  4. Если происходит ошибка, проверьте правильность написания ссылки. Ссылка должна иметь вид: https://vk.com/group_name

Генератор отчётов

  1. Скопируйте ссылку на группу ВКонтакте, отчёт по которой вам нужен

  2. Вставьте ссылку в текстовое окно
  3. Выставьте дату начала периода отчётности и конца периода отчётности. Дата начала отчётности не должна быть раньше конца отчётности

  4. Нажмите кнопку «Отправить»
  5. Отчёт будет находится в папке приложения по пути: smm_parser\app\output
  6. Если происходит ошибка, то:
    1. Возможно, в группе нет постов за указанный период
    2. Проверьте правильность написания ссылки и указанных дат. Ссылка должна иметь вид: https://vk.com/group_name.
    3. Сообщите об ошибке на почту: margl8@yandex.ru
